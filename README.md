# sudoku-game
Created with CodeSandbox

Aplicacion de sudoku con 4 dificultades
- Easy (4x4)
- Medium (9x9)
- Hard (9x9 con menos pistas)
- Insane (16x16)

Si queres saber la respuesta al sudoku, vas a tener que pagarme 0.1 Ether
